--------------------------------------------------------
                    Brick Eater  [x]-----[]
--------------------------------------------------------

This script is to play the awesome game Brick Eater I created
as a python final.

The controls :
-W A S D to move around
-P to pause/unpause game
-A button to continue when you loose
-Q button to quit when you loose

The Objective :
-Like the old snake game, eat the bricks to grow!

Have fun!
